# ユビキタス言語

プロジェクト全体で統一して使用するドメイン用語の定義集です。
コード・ドキュメント・会話のすべてでこの用語を使用してください。

## 珈琲豆

| 用語 | 英語名 | 説明 | 短縮語 |
|------|--------|------|--------|
| 珈琲豆 | coffee bean | 販売・提供の対象となる珈琲豆 | |
| 店舗 | shop | 珈琲豆を提供する店舗 | |
| テイスト | taste | 珈琲豆の味わいの評価軸。「酸味」「苦味」「コク」など | |
| 珈琲リスト | coffee list group | 珈琲豆をグルーピングしたリスト。CSサイトに表示する | |
| 焙煎度 | roast level | 珈琲豆の焙煎の深さ。一般的には8段階だが、本サービスでは浅煎り/中煎り/中深煎り/深煎りの4段階で扱う | |
| 浅煎り | light | 焙煎度。軽い焙煎 | |
| 中煎り | medium | 焙煎度。中程度の焙煎 | |
| 中深煎り | city | 焙煎度。やや深い焙煎 | |
| 深煎り | french | 焙煎度。深い焙煎 | |
| 精製方法 | processing method | 珈琲豆の精製方法 | |
| フリーウォッシュド | fully washed | 精製方法。水洗式の完全版 | FW |
| ウォッシュド | washed | 精製方法。水洗式 | W |
| サーマルショックナチュラル | thermal shock natural | 精製方法。熱衝撃を与えたナチュラル | TSN |
| ナチュラル | natural | 精製方法。自然乾燥式 | N |
| スマトラ式 | wet hulling | 精製方法。半水洗式 | |
| ハニー | honey | 精製方法。半水洗式の一種 | |

## サブスクリプション

| 用語 | 英語名 | 説明 | 短縮語 |
|------|--------|------|--------|
| サブスクリプション契約 | subscription | 顧客がプランに対して結ぶ契約 | サブスク |
| プラン | subscription plan | サブスクリプションの料金プラン | |
| 契約期間 | contract period | サブスクリプション契約の開始日〜終了日 | |
