# Taste 集約

## 概要

珈琲豆のテイスト評価の種別を表す集約。
「酸味」「苦味」「コク」などのテイスト名を管理する。

## ライフサイクル

- **登録**: 管理者が管理画面から登録する
- **更新**: テイスト名の変更

## Taste

### プロパティ

| プロパティ | 型 | 説明 |
|-----------|---|------|
| id | TasteId | テイストID |
| name | String | テイスト名 |

### 不変条件

- nameは必須
- nameはシステム内で一意でなければならない
